Daily Cal Check Instructions for UHSAS during DC3/SEAC4RS

  1. Follow startup instructions for UHSAS
  2. Turn 3-way valve to sample cabin air and remove swage cap, if present
  3. Set up and connect the portable nebulizer to the UHSAS
  4. Add a few ml of the PSL cal check liquid to the nebulizer
  5. Place cap on nebulizer and turn on pump
  6. Examine the UHSAS "histogram" screen. You can adjust the x and y scales to be logarithmic if you like
  7. Look for distinct peaks at 125, 200, 300, and 500 nm
  8. If "excessive particle counts" error message appears, dilute solution 2:1 with distilled water and repeat
  9. Turn off and disconnect nebulizer tube; connect zero air filter to 3-way valve
  10. Check for zero counts on "histogram" page
  11. Stop spectrometer program or disable the "save" button and download most recent (small) file in "UHSAS data" directory to a flash drive
  12. Turn off the UHSAS via rear circuit breaker or AC#1 circuit breaker on rack power distribution box, or re-enable the "save" button and continue running
  13. Disconnect filter and return 3-way valve to sample from the CRD/PAS dryer
  14. Check on/off valve at flow shelf exit
