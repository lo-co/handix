Startup Instructions for the UHSAS

  1. Open on/off valve exiting flow shelf and check 3-way valve at rear of UHSAS
  2. Check water level in sight tubes and add distilled water as needed. Eliminate all bubbles from water line
  3. Engage circuit breaker at rear of UHSAS
  4. Turn on AC#1 breaker on rack power distribution box. Note: do NOT run UHSAS for longer than 1 minute without using the laptop and LabView program
  5. Start "Spectrometer" LabView program
  6. Go to "Controls" tab and check that flows are on. IMPORTANT: check that Laser Temperature setting is 1.23 V and that the measured laser temperature is controlling within 0.04 V of this valueSometimes this does not control and a runaway could damage the laser. If it not controlling, cycle the power. NOTE: AFTER ENTERING A VALUE IN ANY BOX, YOU MUST CLICK OUTSIDE THE BOX TO UPDATE THAT SETTING. PRESSING THE "ENTER" KEY DOESN'T DO IT.
  7. If no measurements are appearing or are not updating there is a COM error:
    • Go to the "Configuration" tab
    • In the password box enter "delt" and press the "enter password" button
    • Select a new COM port, such as COM 9 (COM 3 should be default), and press the "revert mode" button
    • Return to the password box, enter "delt" and press the button again
    • Select COM 3, then press the "revert mode" button
    • Press "commit" button at top right
    • Return to the "Controls" tab and check that values are updating and laser temperature is controlling
  8. Go to "RH control" tab
    1. Enable TEC
    2. Set flow mode to "humidifier" (other option is "bypass")
    3. Set leftmost switch to control to RH
    4. Set RH control value to 85%
    5. You should see temps and RH values begin to control over a few minutes
  9. Go to "histogram" tab
    1. Make sure data acquisition time is set to 1 second
    2. Make sure "run" button is enabled
    3. Make sure "record" button is enabled
    4. You should see a histogram of particle counts that updates at 2 Hz
