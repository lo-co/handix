Signal Connections

The table below defines the actual physical connections of the PAS signals to the data acquisition cards. The headers are defined as below:
  • Signal - description of the signal
  • in/out - whether the signal is acquire (in) or generated (out)
  • voltage - voltage range of the signal
  • # bits - resolution of the signal
  • rate - acquisition/generation frequency
  • Device - physical NI DAQ card that the signal is connected to
  • channels - DAQ defined channel of the signal; this value is used to identify the connection point in software
  • connector pin(s) - the physical connection point of the signal to the card itself
  • DAQmx Task - task used for generation or acquisition of the signal. This can be used to probe the signal in MAX.
  • Position in Task - position of the signal in the DAQmx task relative to the other signals
  • notes - pertinent notes pertaining to the signal
