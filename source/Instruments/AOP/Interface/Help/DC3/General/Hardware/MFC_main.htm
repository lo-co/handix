Mass Flow Controllers

All mass flow controllers (MFCs) communicate on a single bank located in the rack on the starboard side. The bank is connected to COM 1 on the chassis (the DB-9 port located on the PXI controller) and is routed through the PAS side power distribution box. The user may communicate with the MFCs via the Measurement and Automation Explorer (MAX). In order to communicate with the MFCs:

  • The computer hosting the user interface must be connected to the PXI chassis through the ethernet port and the PXI chassis must be powered on.
  • No other program must be attempting to communicate with the port. This means that the PAS/CRDS software must be OFF.
  • The DC power supply must be on.
The following sections describe how to communicate with the Alicat MFCs listed above outside of the main user interface. Note that communication can only proceed through MAX when the user interface is no longer communicating with the MFC bank. Due to the installation, the user interfaces on the individual MFC front panels is not possible and therefore communication must occur through the computer.
