The status flag is a single byte that contains the status of several UI flags:
  1. Auto speaker - speaker switches on fixed duty cycle
  2. Link speaker to filter cycles
  3. Bypass - system is set to bypass filter and sample for calibrations
  4. AutoFilter
  5. O3 - set by user to indicate O3 cals
  6. Filter position
If 0 AND 1 AND NOT (2 OR 4), then the speaker will automatically turn on when flow is switched to filter. If 4 then 0 AND 3 are disregarded.
