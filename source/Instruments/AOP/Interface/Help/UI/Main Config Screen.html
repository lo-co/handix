CRDS Parameters

Exponential Fit Method

These controls set the fitting method for the cells associated with the three different lasers.  Common methods for analyzing exponential traces traditionally include a linear least-squares fit corrected for a baseline offset and the Levenberg-Marquardt non-linear least squares technique.  Due to the number of traces that are fit in a single second (8000 for a pulse rate of 1 kHz), both of these methodologies are insufficient due to available execution time without coadding.  Three different techniques have been outlined in recent literature that provide an efficient methodology for fitting ringdown traces with high accuracy.  These techniques are defined in the available options of the Exponential Fit Method box as DFT-1, DFT-5 and LRS.  Each of these methods requires no baseline for fitting data and is of an order of magnitude faster than the non-linear algorithm of Levenberg-Marquardt.

LRS

Linear regression of the sums, or LRS, is a technique defined in Everest and Atkinson [2008].  LRS is the retangular integration solution of linear least squares, this techniques solves the following linear system

| N SI  St   |  | a0 | = |Sf |
| SI SII StI |  | a1 | = |SfI |
| St StI Stt  |  | a2 | = |Sft |

where N is the numberr of points in the exponential and Sx represents the sum of term x and Sxy represents the sum of the product of two terms at each point evaluated.  I is the running integral of the signal x (evaluated using rectangular integration, or Σ(fi*ti) where f is the value of the exponential at point i and t is the time at point i (in this case, Δt is = 1).  The matrix A is positive definite (apparent from the structure above) and we can use Cholesky decomposition to solve for the vector a.  We can then evaluate τ using this vector:

  1/τ = ln(1-a1)

DFT-1 and DFT-5

The DFT-1 (and -5) methods map the exponential decay into the frequency domain [Mazurenka et al., 2005].  Calculate τ from the first frequency component of the DFT using:

1/τ = 1/Δt*ln(Re(F(k)/Im(F(k)*sin(wk*Δt) + cos(wk*Δt))

where k = 1 and assuming unit spacing in time.
